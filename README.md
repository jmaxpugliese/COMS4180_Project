# COMS4180_Project

### Instructions for installing on a fresh VM:

1. Install pip:
`sudo apt-get install python-pip`

2. Install Python virtual environment: `pip install --upgrade virtualenv`

3. Create new Python 3 project: `virtualenv -p python3 <directory>`

4. Navigate to newly created directory: `cd <directory>`

5. Activate virtual environment: `source bin/activate`
